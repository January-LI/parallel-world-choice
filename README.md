# Parallel World Choice

A terminal-based interactive entertainment project built with Python.

Parallel World Choice combines random fortune-telling experiences with mystery box culture. Users simply enter a random number to receive a personalized daily horoscope and exclusive recommendations, as if unlocking a completely different destiny from another parallel universe.

The project also includes a **Mystery Box Assistant** feature, which uses lucky numbers, keyword resonance, and a random recommendation system to create a more immersive and ritual-like experience when selecting mystery boxes, turning random purchases into a feeling of being "chosen by fate."

## Features

- Daily Fortune System
- Mystery Boxes Fate Assist System (Mystery Box Assistant)
- Random lucky numbers and colors
- File-based history saving

## Files

- `main.py` -> Main menu and interaction
- `fortune_data.py` -> Stores all fortune data
- `fate.py` -> Fortune and fate logic
- `save.json` -> Saved history data

## How to Run

1. Install NumPy:

```bash
pip install numpy
```

2. Run the project:

```bash
python main.py
```
